# Invoice Application Documentation

This directory contains the comprehensive documentation for the Invoice Application system.

## Documentation Structure

### [Entities](entities.md)
Defines the core data entities in the system:
- **EN0001** - User
- **EN0002** - Invoice
- **EN0003** - Item

### [Functions](functions.md)
Describes system functions and features:
- **FN0001** - Authentication
- **FN0002** - Invoice ID generation
- **FN0003** - Invoice due date calculation
- **FN0004** - Filtering
- **FN0005** - Pagination
- **FN0006** - Currency
- **FN0007** - Dark mode

### [Use Cases](use-cases.md)
Details user interactions and workflows:
- **UC0001** - User creates an account
- **UC0002** - User logs in
- **UC0003** - User logs out
- **UC0004** - User recovers the password
- **UC0005** - User creates new invoice
- **UC0006** - User edits invoice
- **UC0007** - User deletes invoice
- **UC0008** - User adds item to invoice
- **UC0009** - User edits item in invoice
- **UC0010** - User removes item from invoice
- **UC0011** - User changes app theme

### [Roadmap](roadmap.md)
Lists planned features and future enhancements:
- Password recovery
- Multiple themes
- Form of invoice payment
- Client catalog
- Multicurrency support
- Invoice soft delete
- Additional invoice state business logic
- Overdue invoice detection
- Tax/VAT support
